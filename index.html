How The Web Works

Web is a collection HTML (Hyper Text Markup Language) pages. Your computer uses the internet and the protocol HTTP to communicate with servers which house the HTML code of webpages.

HTML Markup

HTML relies on the creation of elements using tags that have an opening and a closing.

Tags

HTML relies on tags so that browsers can correctly translate what will be displayed on the page. A good summary of tags tags can be seen here.

Inline vs Block

Level 2

Web Development (Front End)

CSS, HTML, and Java Script are the three primary languages used in web design. HTML provdes structure (like the walls of a house), CSS provides style (decorations , paint, etc), while Java Script provides interactive components (like the garage door opener). In Chrome, clicking on the 3 stacked lines, then tools, then development tools will show you the structure of a page. It is best to think of elements being inside of an invisible box.

HTML's Tree Like Structure

Boxes are nested inside other boxes. Each box provides the framwork for the for the text/video/image that you see on the screen. For example, HTML->Body->h1->text. HTML is thr trunk, body is a branch with other braches sprouting from it. The tip of the branch is what will be displayed inside that box on the screen by the browser. In other words, the tags represent the branches of a tree. The Document Object Model (DOM) is a page's tree structure.

When designing a site, start with a collection of boxes then fill them in. Each box should be defined by a div tag.

When To Use The Tools

Scratchpad is the easiest to use when writing HTML and CSS however, it is the least powerful. Codepen can be used to write and share code.

Use CSS....don't Repeat Yourself!

Cascading Style Sheets (CSS) are used to add styles to classes in HTML code. This is language is essential in ensuring that programmers do not repeat themselves and can remain efficient.It does this by applying rules to elements/classes/descriptions.

Inheritance

This concept centers around the fact that properties are applied to specific elements and its descendants.

The Box Model

Margin -> Border -> Padding -> Content. Pro tip: Set boxes as a percentage so they will change along with a user's screen size.
